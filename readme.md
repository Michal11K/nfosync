# NfoSync

> Your metadata. Your rules.

NfoSync keeps your TinyMediaManager NFO files synchronized with live metadata providers while preserving NFO files as the single source of truth.

---

## Why?

Most media servers gradually become the owner of your metadata.

NfoSync takes the opposite approach.

Your NFO files remain authoritative.

Applications like Jellyfin, Kodi and Emby simply consume them.

---

## Features

- ⭐ Update IMDb ratings
- ⭐ Update Trakt ratings
- ⭐ Update TMDb ratings
- ⭐ Synchronize TV show status
- ⭐ Synchronize collection metadata
- ⭐ Refresh Jellyfin items
- ⭐ Preserve all user metadata
- ⭐ TinyMediaManager compatible

---

## Philosophy

NfoSync never generates NFO files.

NfoSync never overwrites static metadata.

NfoSync only synchronizes dynamic metadata.
